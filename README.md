# SN68 NOVA submissions

Drand-timelock-encrypted NOVA submission blobs.

Each blob lives at `submissions/<target>/<hash20>.txt` and is referenced by an
on-chain `set_commitment(netuid=68)` extrinsic from the miner's hotkey. The
validator fetches the blob via raw.githubusercontent.com after the reveal
window opens and `literal_eval`s it into a list of `(rxn:N:id1:id2:id3, sequence)`
candidates per the [metanova-labs/nova](https://github.com/metanova-labs/nova)
protocol.

This repo is generated by `bittensor/sn68-nova/offline-validation/src/submit_live.py`.
